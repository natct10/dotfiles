# Vim setup

## Pathogen

```sh
mkdir -p ~/.vim/autoload ~/.vim/bundle && \
curl -LSso ~/.vim/autoload/pathogen.vim https://tpo.pe/pathogen.vim
```

## NERDTree

```sh
git clone https://github.com/scrooloose/nerdtree.git ~/.vim/bundle/nerdtree
```
